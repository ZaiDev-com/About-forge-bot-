About FORGE BOT

Your Telegram Weather Assistant

FORGE BOT is an intelligent Telegram bot that provides accurate weather forecasts and environmental data directly in your chat. No need to switch between apps - get all your weather information where you already communicate.

Using advanced Water API technology, FORGE BOT delivers hyper-accurate weather predictions, real-time updates, and severe weather alerts right in Telegram.

How to Use

Simply start a chat with FORGE BOT and use these commands:

  • /start - Initialize the bot
  • /weather [city] - Get current weather
  • /forecast [city] - 5-day weather forecast
  • /alerts - Severe weather warnings
  • /help - Show all commands
